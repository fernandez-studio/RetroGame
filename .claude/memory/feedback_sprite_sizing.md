Never make the damsel (or any other humanoid character) smaller than the gunslinger sprites. The cowboy is 9×14 game pixels — all characters must match this size.

**Why:** Lower resolution makes the character look inhuman and unrecognizable. User rejected multiple smaller damsel sprites for this reason.

**How to apply:** When creating or redesigning any humanoid sprite, check COWBOY dimensions first (currently 9×14) and use the same grid size. Do not downscale "to fit" — the arena is large enough.
